Overview
========

RoSHI combines:

- **9 body-worn IMU trackers** for robust local pose sensing
- **Project Aria glasses** for egocentric video and global motion context

System Output
-------------

- Full 3D body pose (SMPL format) in a metric global coordinate frame
- Egocentric RGB video
- Globally consistent root trajectory

Key Features
------------

- **Portable**: fully wearable and self-contained
- **Low-cost**: under $500 in tracker hardware, excluding Aria
- **Quick calibration**: a 20-40 second iPhone video is enough
- **In-the-wild**: no external camera rig is required
