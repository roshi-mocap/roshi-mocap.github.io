# RoSHI Project Page

This is the project webpage for **RoSHI: Robust Sparse-IMU Human Pose Estimation**.

Built from the [Nerfies](https://github.com/nerfies/nerfies.github.io) website template (CC BY-SA 4.0).

## Usage

Open `index.html` in a browser, or deploy via GitHub Pages:

1. Create a repo named `<username>.github.io` (or use a project repo with GitHub Pages enabled).
2. Push this directory to the repo.
3. Your page will be live at `https://<username>.github.io/`.

## Customization

Search for `TODO` comments in `index.html` to find all placeholder content that needs updating:

- **Title & subtitle** — project name and one-line description
- **Authors & affiliations** — names, links, institutions
- **Publication links** — Paper PDF, arXiv, YouTube video, GitHub code
- **Teaser** — hero image or video (`static/images/` or `static/videos/`)
- **Abstract** — paper abstract text
- **Method & Results** — figures, tables, videos
- **BibTeX** — citation entry

Place media assets in `static/images/` and `static/videos/`.
