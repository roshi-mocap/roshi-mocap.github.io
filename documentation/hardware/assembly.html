Assembly#

This page covers how to assemble the RoSHI wearable suit.

Note

Detailed assembly instructions with photos are coming soon.

Tracker Assembly#

  1. Solder the BNO085 breakout to the ESP32 via I2C (SDA/SCL)

  2. Connect the LiPo battery to the ESP32 power input

  3. Mount the assembled electronics in the 3D-printed enclosure

  4. Attach the AprilTag (42 mm, Tag36h11) rigidly to the enclosure face

Suit Integration#

  1. Attach each tracker to the corresponding body location using Velcro straps

  2. Ensure the AprilTag face is visible and not occluded by clothing

  3. Wear the Project Aria glasses and pair via the Aria companion app
