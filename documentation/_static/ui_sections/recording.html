#Element (Recording State Changes)
11
Recording Duration
Shows elapsed recording time in MM:SS format with red background. Updates every 0.1 seconds. Only visible while recording is active.
12
Live Detection Counts
During recording, each chip shows the current detection count (replacing the tag ID). Progress bar fills as detections accumulate. Colors change: red = below target, green = target reached. Counts saturate at the target value.
Record Button (Stop Mode)
While recording, the button turns full red with a white square (stop icon). Tap to stop recording. After stopping, files are automatically transferred to the receiver.
Low Detection Warning
If recording is stopped before all tags reach the target count, an alert dialog appears: "Some tags did not reach the target of N detections. The recording has been saved anyway."
