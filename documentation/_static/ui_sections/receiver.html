14
Connection Status
Shows whether the app is currently connected to the RoSHI receiver. Green = connected, Red = disconnected. The app uses Bonjour/mDNS for automatic discovery.
15
Receiver Address
Tap "Current Address" to expand and see/edit the connection details. Shows the IP and port. Tap the eye icon to reveal, tap again to hide. By default the address is hidden.
16
Manual IP/Port Entry
If automatic discovery fails, manually enter the IP address (e.g., 10.103.76.0) and port (default: 50000) of the computer running 01_receiver.py. Tap Save to apply.
